Welcome to my AWS Cloud Journey repository.

This repository documents my hands-on learning as I transition into Cloud Engineering. Rather than simply studying AWS concepts, I build practical projects, document my implementation process, record challenges, and reflect on what I learn.

## Purpose

The goal of this repository is to:

* Build practical AWS cloud experience
* Develop a portfolio of real-world projects
* Reinforce theoretical concepts through hands-on practice
* Track my progression from beginner to cloud engineer
* Demonstrate my problem-solving and documentation skills

## Learning Philosophy

I learn best by comparing AWS services with technologies I already know from my homelab.

Examples include:

| AWS             | My Existing Knowledge        |
| --------------- | ---------------------------- |
| EC2             | Proxmox Virtual Machine      |
| EBS             | Virtual Disk                 |
| Security Groups | Firewall Rules               |
| VPC             | Home Network                 |
| IAM             | Active Directory Permissions |

This repository focuses on understanding *why* AWS services exist, *when* to use them, and *how* they integrate into real-world cloud architectures.
